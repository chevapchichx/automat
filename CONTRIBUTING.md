# Инструкция по добавлению новых номеров

## Добавление фотографий

1. Поместите изображение в папку:
   ```
   app/src/main/res/drawable-nodpi/
   ```

2. Используйте формат имени: `room_XXX.jpg` (где XXX - номер комнаты)
   - Пример: `room_303.jpg`, `room_404.jpg`

3. Поддерживаемые форматы: JPG, PNG, WEBP

## Добавление номера в БД

Откройте файл `DatabaseHelper.kt` и в методе `onCreate()` добавьте строку:

```kotlin
insertRoom(
    db,
    "303",                           // Номер комнаты
    2,                               // Вместимость (человек)
    1800.0,                          // Цена за сутки (₽)
    "Стандартный номер",             // Описание
    1,                               // Доступность (1 - да, 0 - нет)
    "WiFi, ТВ, Кондиционер",         // Удобства
    "room_303"                       // Имя файла изображения (без расширения)
)
```

## Обновление версии БД

После добавления новых номеров увеличьте версию БД в `DatabaseHelper.kt`:

```kotlin
private const val DB_VERSION = 4  // Было 3
```

И добавьте миграцию в метод `onUpgrade()`:

```kotlin
if (oldVersion < 4) {
    // Добавить новые номера
}
```

## Удаление старой БД для тестирования

Если нужно пересоздать БД с нуля (удалит все данные):

1. В Android Studio: `Device File Explorer`
2. Перейдите в `/data/data/com.example.automat/databases/`
3. Удалите файл `rooms.db`
4. Перезапустите приложение

Или через adb:
```bash
adb shell rm /data/data/com.example.automat/databases/rooms.db
```
